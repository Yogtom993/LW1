# Лабораторна робота №1

## Тема

HTML5 та CSS.

## Назва проєкту

**TechStore — каталог продукції компанії**

## Мета роботи

Вивчити стандарт HTML5 та CSS, способи створення вебсторінок і підключення каскадних таблиць стилів.
Набути практичних навичок розробки багатосторінкового вебсайту з використанням основних HTML5-тегів, CSS-оформлення, таблиць, списків, форм, зображень, фреймів та мультимедійних елементів.

## Опис проєкту

У межах лабораторної роботи було створено навчальний вебсайт **TechStore** — каталог продукції компанії, яка продає електроніку та комп’ютерну техніку.

Сайт складається з трьох взаємопов’язаних сторінок:

* головна сторінка;
* сторінка каталогу товарів;
* сторінка контактів і форми замовлення.

На сайті використано HTML5-структуру, єдиний CSS-файл для оформлення сторінок, таблиці з товарами, списки, зображення, навігацію, форму, iframe, аудіо та відео.

## Виконані завдання

### Блок №1

1. **Розроблено вебсайт із трьома взаємопов’язаними сторінками**
   Створено сторінки `index.html`, `catalog.html` та `contacts.html`.
   У структурі сторінок використано HTML5-теги: `<header>`, `<main>`, `<footer>`, `<aside>`, `<article>`, `<section>`.

2. **Використано основні HTML-елементи**
   На сторінках додано заголовки, абзаци, зображення, посилання, списки, таблиці, засоби форматування тексту, фон сторінки та нижній колонтитул.

3. **Додано мета-теги**
   У HTML-документах використано метатеги для кодування сторінки, адаптивності та опису сайту.

4. **Підключено єдиний CSS-файл**
   Для всіх сторінок використано один файл `style.css`, у якому описано зовнішній вигляд сайту.

5. **Використано CSS-селектори, класи та ідентифікатори**
   У стилях застосовано селектори тегів, класи, ID-селектори, налаштування кольорів, шрифтів, фону, рамок, відступів і курсора.

6. **Описано структуру вебсайту**
   Створено структуру сайту TechStore з головною сторінкою, каталогом і сторінкою контактів.

### Блок №2

1. **Додано навігаційне меню**
   На сторінках створено блок `<nav>` з посиланнями для переходу між сторінками сайту.

2. **Реалізовано фрейм**
   На сторінці контактів додано `iframe`, у якому відображається сторінка каталогу.

3. **Додано аудіо та відео**
   На сторінці контактів використано HTML5-теги `<audio>` і `<video>` з елементами керування.

### Блок №3

1. **Використано градієнт, тінь і прозорість**
   У CSS-оформленні застосовано градієнтний фон, тіні для блоків і прозорість окремих елементів.

## Структура проєкту

```text
LR1_TechStore/
├── index.html
├── catalog.html
├── contacts.html
├── style.css
├── images/
├── media/
└── README.md
```

## Опис файлів

* `index.html` — головна сторінка сайту TechStore;
* `catalog.html` — сторінка каталогу товарів із таблицею та картками продукції;
* `contacts.html` — сторінка контактів, форми замовлення, фрейму, аудіо та відео;
* `style.css` — єдиний CSS-файл для оформлення всіх сторінок;
* `images/` — папка із зображеннями товарів та графічними елементами;
* `media/` — папка з аудіо- та відеофайлами;
* `README.md` — опис проєкту.

## Як запустити проєкт

1. Завантажити або клонувати репозиторій.
2. Відкрити папку проєкту.
3. Запустити файл `index.html` у браузері.
4. Для переходу між сторінками використовувати навігаційне меню.
