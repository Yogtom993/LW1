# Лабораторна робота №1

## Тема
HTML5 та CSS.

## Проєкт
TechStore — навчальний вебсайт-каталог продукції компанії.

## Виконані завдання
Для акуратності залишено рівно потрібну кількість завдань для оцінки 5:

- Блок 1: виконано 6 завдань із 6;
- Блок 2: виконано 3 завдання на вибір;
- Блок 3: виконано 1 завдання на вибір.

## Структура
```text
LR1_TechStore/
├── index.html
├── catalog.html
├── contacts.html
├── style.css
├── images/
└── README.md
```

## Сторінки
- `index.html` — головна сторінка сайту;
- `catalog.html` — каталог товарів;
- `contacts.html` — контакти, форма замовлення та iframe.
