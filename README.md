# Лабораторна робота №1

## Тема

HTML5 та CSS.

## Назва проєкту

**TechStore — каталог продукції компанії**

## Мета роботи

Вивчити стандарт HTML5 та CSS, способи виконання базових операцій і особливості створення вебсторінок з використанням каскадних таблиць стилів.

## Опис проєкту

У межах лабораторної роботи створено навчальний вебсайт **TechStore** — каталог продукції компанії, яка продає електроніку та комп’ютерну техніку.

Сайт складається з трьох взаємопов’язаних сторінок:

* `index.html` — головна сторінка;
* `catalog.html` — сторінка каталогу товарів;
* `contacts.html` — сторінка контактів, форми, фрейму, аудіо та відео.

## Виконані завдання за методичкою

### Блок №1

#### Завдання №1

**Розробити вебсайт із трьома взаємопов’язаними сторінками, використовуючи елементи розмітки `<header>`, `<main>`, `<footer>`, `<aside>`, `<article>`, `<section>`.**

Для виконання завдання створено три HTML-сторінки: `index.html`, `catalog.html` та `contacts.html`.
На сторінках використано семантичні елементи HTML5: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` та `<footer>`.

#### Завдання №2

**Використати основні HTML-теги, кольори, абзаци, заголовки H1–H6, картинки, посилання, списки, таблиці, фон сторінки, форматування тексту та footer.**

На сайті додано заголовки, абзаци, зображення товарів, гіперпосилання між сторінками, нумеровані та марковані списки, таблицю з товарами, форматування тексту, фон сторінки та нижній колонтитул.

#### Завдання №3

**Включити в сайт відомі мета-теги.**

У HTML-документи додано мета-теги:

* `charset`;
* `viewport`;
* `description`;
* `keywords`;
* `author`.

#### Завдання №4

**Доповнити сайт єдиним стилем для всіх сторінок засобами CSS.**

Для оформлення всіх сторінок створено єдиний файл `style.css`.
У ньому налаштовано кольори, шрифти, фон, відступи, рамки, таблицю, картки товарів, форму та навігацію.

#### Завдання №5

**Розглянути селектори, класи, ID-селектори, шрифти, кольори, фон, вирівнювання, списки, рамки та cursor.**

У CSS використано селектори тегів, класи, ID-селектори, властивості для шрифтів, кольорів, фону, вирівнювання, списків, рамок і курсора.

#### Завдання №6

**Намалювати структуру вебсайту.**

Структуру сайту TechStore описано через сторінки:

* головна сторінка;
* каталог товарів;
* контакти та форма замовлення.

### Блок №2

#### Завдання №1

**Додати до сайту навігаційне меню.**

На всіх сторінках створено навігаційне меню `<nav>`, яке містить посилання на головну сторінку, каталог і контакти.

#### Завдання №2

**Реалізувати фрейм.**

На сторінці `contacts.html` додано фрейм `<iframe>`, у якому відображається сторінка каталогу товарів.

#### Завдання №3

**Додати аудіо та відео з елементами керування.**

На сторінці `contacts.html` додано HTML5-елементи `<audio>` і `<video>` з атрибутом `controls`.

### Блок №3

#### Завдання №1

**Використати CSS-властивості для градієнта, тіні та прозорості.**

У CSS-оформленні використано градієнтний фон, тіні для блоків і прозорість окремих елементів.

## Структура проєкту

```text
LR1_TechStore/
├── index.html
├── catalog.html
├── contacts.html
├── style.css
├── images/
├── media/
└── README.md
```

## Опис файлів

* `index.html` — головна сторінка сайту TechStore;
* `catalog.html` — сторінка каталогу товарів;
* `contacts.html` — сторінка контактів, форми, фрейму, аудіо та відео;
* `style.css` — єдиний файл стилів для всіх сторінок;
* `images/` — папка із зображеннями товарів;
* `media/` — папка з аудіо- та відеофайлами;
* `README.md` — опис проєкту.

## Як запустити проєкт

1. Завантажити або клонувати репозиторій.
2. Відкрити папку проєкту.
3. Запустити файл `index.html` у браузері.
4. Для переходу між сторінками використовувати навігаційне меню.
