# Лабораторна робота №3

## Тема

Динамічний HTML. Форми. Основи JavaScript.

## Назва проєкту

**TechStore — каталог продукції компанії**

## Мета роботи

Вивчити основи JavaScript, DOM-модель документа та способи керування елементами вебсторінки.
Набути практичних навичок роботи з подіями, таймерами, зміною вмісту сторінки, формами та локальним збереженням даних.

## Опис проєкту

У межах лабораторної роботи було доповнено сайт **TechStore** інтерактивними елементами JavaScript.
Сайт має сторінки каталогу техніки, демонстрацію динамічних елементів, форму та блоки для виконання завдань лабораторної роботи.

JavaScript використовується для:

* зміни тексту та розміру шрифту;
* переміщення зображення на сторінці;
* роботи з абзацами;
* створення текстового годинника;
* поступового витирання тексту;
* зміни кольору квадрата;
* визначення координат миші та коду клавіші;
* зміни розміру тексту кнопками A+ / A-;
* приховування рекламного блоку;
* збереження нотаток у `localStorage`.

## Виконані завдання

### Блок №1

1. **Функція для виведення тексту з різним розміром шрифту**
   Реалізовано функцію, яка приймає текст і розмір шрифту, після чого виводить результат на сторінку.

2. **Рухоме зображення**
   Додано рухомий елемент із зображенням ноутбука. Він змінює положення кожну секунду за допомогою `setInterval`.

3. **Зміна розміру всіх абзаців**
   Реалізовано пошук усіх тегів `<p>` через `getElementsByTagName` і зміну їхнього розміру через `setAttribute`.

4. **Текстовий годинник**
   На сторінці працює годинник, який оновлюється щосекунди.

5. **Поступове витирання тексту**
   Реалізовано ефект поступового зникнення частини документа за допомогою таймера.

### Блок №2

1. **Список із п’яти кольорів і квадрат**
   Користувач вибирає колір зі списку, після чого колір квадрата змінюється.

2. **Координати миші**
   На сторінці відображаються поточні координати курсора.

3. **Код натиснутої клавіші**
   Після натискання клавіші виводиться її код.

4. **Кнопки A+ / A-**
   Реалізовано зміну розміру тексту за допомогою кнопок. Вибір користувача зберігається.

### Блок №3

1. **Рекламний блок із кнопкою “прибрати”**
   Додано промо-блок, який можна закрити.

2. **Блокнот із записами**
   Створено простий блокнот, у якому записи зберігаються через `localStorage`.

## Структура проєкту

```text
LR3_TechStore/
├── index.html
├── catalog.html
├── contacts.html
├── style.css
├── script.js
├── images/
└── README.md
```

## Опис файлів

* `index.html` — головна сторінка сайту з JavaScript-завданнями першого блоку;
* `catalog.html` — сторінка каталогу TechStore та інтерактивних елементів другого блоку;
* `contacts.html` — сторінка контактів, форми, рекламного блоку та блокнота;
* `style.css` — файл стилів для оформлення сторінок;
* `script.js` — файл JavaScript-коду;
* `images/` — папка із зображеннями та графічними елементами;
* `README.md` — опис проєкту.

## Як запустити проєкт

1. Завантажити або клонувати репозиторій.
2. Відкрити папку проєкту.
3. Запустити файл `index.html` у браузері.
4. Для переходу між сторінками використовувати навігаційне меню.
